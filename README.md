# Kalshi Scout — Discord Bot

A lightweight Discord bot for Kalshi traders:
- Global slash commands (works in **any** server)
- Finds **price spikes**, **top movers**, **volume + Δ**
- **Watchlists** with threshold alerts
- Prices shown in **cents** (e.g., `65¢`)
- Optional tiny health server for hosts that require a port
